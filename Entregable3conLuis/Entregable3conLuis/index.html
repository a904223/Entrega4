NAME: Trabajadores : a working shift timetable.

AUTHOR: Aitor Aguirrebeña Ollo

SHORT DESCRIPTION:

  • This function will allow the user to check each employee's work shifts and also to add off days if required.

Introduce the following date to test it: 08/03/2017

EXPLANATION: When the user accesses the PedirTrabajadores.html form, it will be required to introduce a date. Once this is done, the servlet will get those values and print an interactive work shift table, by clicking the submit button and will print a calendar with the employees' working shifts, by getting them from the database. Only a week will be printed, so that all the shifts can fit. There is a draggable button to set a day or days off, which by dropping it on a certain day it automatically updates the work shift. Drag and drop the Day Off to change the Database information, sending a request to the servlet Save_Shifts, via AJAX and getting the response.

This whole functionality includes two functions:
  • Trabajadores: it prints the shifts and the calendar.
  • Save_Shifts: once the "Day Off" drop off is done, it inserts in the database the selected day, and it sets it off for the chosen employee.
